### Notes for Layer 0
Implented Tap Dance keys for many of the keys typically modified with CTRL. They are enabled with a macro using the same number, so Macro 3 goes with Tap Dance 3. In the image above, italics is for the hold action. Here is the list:
- M3 Undo Z
- M4 Cut X
- M5 Copy C
- M6 Paste V
- M7 Bold B
- M8 Underline U
- M9 Italics I
- M10 Open O
- M11 Print P
- M12 New N
  
Tap Dance 13 is for Esc or Super key when held down

Tap Dance 14 - 17 are the modifiers CTRL, Shift, Alt, and Super when held down. In the image above, italics is for the Hold action.

### Notes for Layer 1
Macro 1 (Alt+Space) brings up the command pallete from Windows Power Toys 

Macro 0 (Mod+C) switches to Outlook using Win+2, then switches to Calendar with Ctrl+2. This works for New Outlook only

Macro 2 (CTRL+D, Space, CTRL+T) prints the date, a space, then the time in Obsidian

F9 (mapped to Q) is the AutoHotkey keyboard shortcut for toggling mute on a Teams call
