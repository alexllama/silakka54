# Silakka54 keyboard layout
File for programming the silakka54 keyboard via the Vial [website](https://vial.rocks/).

Current layout

<img width="873" height="840" alt="image" src="https://github.com/user-attachments/assets/bfe74dd4-1c46-418a-985d-306517f16f3e" />

In layer 0 I added Tap Dance keys for the rightmost column. In the image above, italics is for the Hold action.

And the following on Layer 1:
- F9 is the AutoHotkey keyboard shortcut for toggling mute on a Teams call
- M0 - Mod+C will switch to Outlook using Win+2, then switches to Calendar with Ctrl+2. This works for New Outlook.
- M1 - Mod+Enter is the combination Win+Alt+Space to trigger the PowerToys Command Palette (Windows only). Not used currently
- M2 - Mod+D will print the date, a space, then the time in Obsidian using CTRL+D, Space, CTRL+T
