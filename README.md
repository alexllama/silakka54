### Notes for Layer 0
Implented Tap Dance keys for many of the keys typically modified with CTRL. In the image above, italics is for the hold action. Here is the list:
- TD23 Undo Z
- TD24 Cut X
- TD25 Copy C
- TD26 Paste V
  
Tap Dance 0 is for Esc or Super key when held down

Tap Dance 28 - 31 are the modifiers CTRL, Shift, Alt, and Super when held down. In the image above, italics is for the Hold action.

### Notes for Layer 1
Macro 1 (Alt+Space) brings up the command pallete from Windows Power Toys 

Macro 0 (Mod+C) switches to Outlook using Win+2, then switches to Calendar with Ctrl+2. This works for New Outlook only

Macro 2 (CTRL+D, Space, CTRL+T) prints the date, a space, then the time in Obsidian

Macro 3 (Win+Alt+K) toggles muting the microphone in Windows at a global level
